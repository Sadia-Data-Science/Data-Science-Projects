Project Summary

This project analyzes the U.S. Census Income dataset using machine learning to predict whether individuals earn more than $50,000 annually. We evaluate models like Random Forest, XGBoost, KNN, and Logistic Regression, and explore feature importance and socioeconomic patterns in income distribution.
