Predicting Income Level Using Census Data

A Comparative Analysis of Machine Learning Models

This project analyzes the U.S. Census Income dataset using machine learning to predict whether individuals earn more than $50,000 annually. We used a robust data preprocessing pipeline and evaluated multiple classifiers — Random Forest, XGBoost, KNN, Decision Tree, and Logistic Regression — using performance metrics like accuracy, precision, recall, and AUC.

Through dimensionality reduction, feature selection, and class balancing, we uncovered how factors such as education, marital status, and age shape income predictions. Random Forest emerged as the top performer with an AUC of 0.93.
